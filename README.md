# 1vsALL (Godot)

Небольшая 2D-игра на **Godot 4.2** (рендер: **GL Compatibility**).

## О проекте

**1vsALL** - игра с аренами, противниками и боссами, прокачкой и игровыми режимами.

- **Точка входа (main scene)**: `res://menu.tscn`
- **Язык**: GDScript
- **Локализация**: RU/EN (`res://mats/trs/`)
- **Ключевые автозагрузки (autoload)**:
  - `gm` - глобальные данные игры (режимы, список врагов/боссов, карты/арены, статы, таблицы прокачки/предметов) - `res://mats/autoload/game.gd`
  - `fnc` - общие функции/утилиты и доступ к нодам текущей сцены - `res://mats/autoload/functions.gd`
  - `sls` - простая система сохранений в JSON - `res://mats/autoload/sls.gd`

## Управление (по Input Map)

- **Движение**: WASD / стрелки
- **Атака**: ЛКМ или `Z`
- **Ролл/уклонение**: ПКМ или `X` / `Space`
- **Меню/выход**: `Esc`

## Как запустить

1. Открыть проект в Godot **4.2+**.
2. Нажать F5 или запустить сцену `menu.tscn` (она же выставлена как main scene в `project.godot`).

## Структура проекта (крупными блоками)

- `menu.tscn`, `settings.tscn` - меню и настройки
- `mats/player/` - игрок/персонаж
- `mats/enemys/`, `mats/bosses/` - враги и боссы
- `mats/lvls/` - арены/уровни
- `mats/UI/` - UI (в т.ч. карта/пауза/панели)
- `mats/items/`, `mats/contents/`, `mats/statuses/` - предметы/контентные сущности/статусы
- `mats/imgs/`, `mats/sounds/`, `mats/font/`, `mats/shaders/` - ассеты

## Сохранения

Сохранения пишутся в JSON (см. `res://mats/autoload/sls.gd`). При старте создаётся директория `saves/` рядом с проектом/в рабочей директории запуска.
